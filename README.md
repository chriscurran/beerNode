## beerNode v1.0
### Requirements

	- Node.js 
	- 1-Wire filesystem
	
How to setup the Raspberry Pi with Node.js and 1-Wire: [Wiki](https://github.com/chriscurran/beerNode/wiki/Raspberry-Pi).
	

### Install

	- # git clone https://github.com/chriscurran/beerNode.git
	- # cd beerNode
	- # npm install
